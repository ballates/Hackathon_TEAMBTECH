Description de la CAH

La CAH a pour objectif de construire une hirarchie sur les individus et se prsente sous forme dun dendogramme. Cette classification permet de regrouper des individus dans des classes les plus homognes possibles partir dun jeu de donnes individus \(\times\) variables. Cette mthode ncessite de choisr une distance entre individus, ou plus gnralement une dissimilarit et un indice daggration entre classes. La distance choisie est la distance euclidienne et lindice daggrgation celui de Ward. lindice daggrgation de Ward est un critre fond sur linertie). Ce qui nous permet denchainer ACP et CAH.

La demarche est implemente dans la fonction HCPC du package FactoMineR pour les situations o on sintresse au critre de Ward. On effectue ensuite la CAH sur les coordonnes des individus sur les \(2\) premiers axes factoriels. Largument \(ncp=2\) de la fonction PCA permet de spcifier le nombre de composantes retenues.

res.pca <- PCA(dfRaw, quali.sup = c(1:3, 19), ncp = 2, graph = F)
res.hcpc <- HCPC(res.pca, consol = FALSE, graph = FALSE)
